# 16进制转为2进制

我们在看源码的时候总能看见一个16进制的数字

```java
static final int HASH_BITS = 0x7fffffff;   // int_max
```

可以通过 8421 的方式 进行 快速转换

1111 = 8 + 4 + 2 + 1 = 15 =F
1110 = 8 + 4 + 2 + 0 = 14= E
1101 = 8 + 4 + 0 + 1 = 13= D
1100 = 8 + 4 + 0 + 0 = 12 =C
1011 = 8 + 0 + 2 + 1 = 11= B
1010 = 8 + 0 + 2 + 0 = 10 =A
1001 = 8 + 0 + 0 + 1 =9 =9
……
0001 = 0 + 0 + 0 + 1 = 1= 1
0000 = 0 + 0 + 0 + 0 = 0= 0

>int 4字节 32位  首位为符号位   所以 0x7fffffff 就可以理解了 代表 int 类型的最大值
>
>7 = 0111    f = 1111
